This is a project - realtime strategy game.
It will be web based app. Serverless. User will play against computer.
There will be following units:
1. Peasant
2. Swordsman
3. Archer

Resources:
gold (gold mine)
wood

Gold allows to train units
Wood and Gold allow to build structures and (this is important feature: build arrows)
Yes, in this game will be limited number of arrows. Arrows will be make in the Arrow Building from Wood. Peasants will put wood in the Arrow Building and take Arrows and bring them to the Archers.
Focus on functionality, not on design. Use circles to display peasents, squares for swordsmen, triangles for archers. Red color for human player, blue color for computer player.
